# 🩺 Uyir AI – Intelligent Bilingual Healthcare Assistant

## 📌 Overview

Uyir AI is an AI-powered bilingual (English/Tamil) healthcare assistant designed to make healthcare support simple, secure, and accessible. It uses Google Gemini AI to analyze medical reports, assess symptoms, provide medicine information, and offer personalized health guidance through an easy-to-use web application.

---

## 🚀 Features

- 🤖 AI-powered Medical Report Analysis
- 📄 PDF, Image & Text Report Analysis
- 🧠 AI Health Twin for Personalized Health Monitoring
- 🎤 Voice-based Symptom Checker
- 🌐 English & Tamil Language Support
- 💊 Medicine Information Lookup
- ⏰ Medicine Reminder
- 💧 Water Reminder
- 🚨 Emergency First Aid Guidance
- 🔐 Secure User Authentication
- 📊 Health History Dashboard

## 🎯 Problem Statement

Many people struggle to understand medical reports, identify symptoms, and access reliable healthcare information quickly. Language barriers and lack of immediate medical guidance make healthcare more difficult.

---

## 💡 Solution

Uyir AI provides an intelligent bilingual healthcare platform that analyzes reports, assists users with symptoms, offers medicine information, and delivers personalized AI-powered health support.
